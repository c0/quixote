# Quixote

A fast, keyboard-driven macOS tool for enriching structured data files with LLMs.

## Get Started

```sh
make setup      # install xcodegen (once)
make generate   # generate Quixote.xcodeproj
make dev        # build and launch the app
make site-dev   # run the Astro site locally at localhost:4321
make site-build # build the Astro site
```
