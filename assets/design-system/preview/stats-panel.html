Why this shape

Dense & monospace. Every value is tabular-nums so numbers align across refreshes. No rounded card nesting; the bar sits flush with the table above it.

One accent. Blue fills the progress bar; red only appears when failed > 0. Semantic dot for RUNNING / PAUSED / FAILED matches the existing status palette.

Progressive disclosure. Bar always visible while running; Grid opens on Details for per-model breakdown, token split, and retry-failed. Keeps table real estate intact 90% of the time.

Vertical-slice metrics. Not a dashboard. Every metric is something you'd watch mid-run: am I ahead of ETA? is any model degraded? where are the failures?
